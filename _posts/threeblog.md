## Contexto | Context

Para mantener un perfil tecnico competitivo en ciberseguridad, estableci un entorno de practica continua basado en laboratorios Docker y administracion avanzada de Linux.

To maintain a competitive cybersecurity profile, I built a continuous practice environment using Docker-based labs and advanced Linux administration.

## Objetivo | Goal

Traducir teoria de seguridad a ejecucion tecnica aplicable en contextos de produccion: deteccion de riesgo, mitigacion y fortalecimiento operativo.

Translate security theory into execution suitable for production contexts: risk detection, mitigation, and operational hardening.

## Alcance Tecnico | Technical Scope

1. Despliegue de escenarios en Dockerlabs para pruebas ofensivas/defensivas.
2. Evaluacion de vulnerabilidades, incluyendo inyeccion SQL en entornos controlados.
3. Documentacion de hallazgos y recomendaciones de mitigacion.
4. Hardening y administracion de Linux en Fedora, Arch Linux, Pop!_OS y Garuda.
5. Uso de terminologia y enfoque EDR/XDR para monitoreo y respuesta.

1. Deployed Dockerlabs scenarios for offensive/defensive testing.
2. Assessed vulnerabilities, including SQL injection in controlled environments.
3. Documented findings and mitigation recommendations.
4. Applied Linux hardening across Fedora, Arch Linux, Pop!_OS, and Garuda.
5. Used EDR/XDR concepts for monitoring and response mindset.

## Resultados | Results

- Mayor capacidad para identificar vectores de ataque y debilidades de configuracion.
- Mejor criterio para priorizar mitigaciones de alto impacto.
- Fortalecimiento de competencias en sistemas Linux y seguridad operativa.
- Transferencia de aprendizaje a escenarios reales de soporte e infraestructura.

- Stronger ability to identify attack vectors and configuration weaknesses.
- Better judgment to prioritize high-impact mitigations.
- Improved Linux systems and operational security skills.
- Effective transfer of learning to real support and infrastructure scenarios.

## Valor Profesional | Professional Value

Este caso consolida mi perfil como profesional hibrido: desarrollo Full Stack con enfoque defensivo, administracion de sistemas y capacidad de respuesta ante riesgos tecnicos.

This case reinforces my hybrid profile: Full Stack development with a defensive mindset, systems administration, and practical response capability to technical risks.
